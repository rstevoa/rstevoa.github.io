If you don't know how to play Power Quest, the gist is it's an RPG / fighter game that didn't sell well - I imagine because it was so grindy. Anyway, in the game you pick one of five robot models, each of which with three abilities, the ability to counterattack, and the options for a powerup which will heal you and a powerup which will fill your special attack meter. Which of course means you have special attacks in that game - two per character.
The game has a control scheme somewhat similar to Street Fighter. For example, to use Max's abilities:
↓ → A => Projectile
→ ↓ A => Uppercut
↓ ← A => Forward Punch
← ↓ A => CounterAttack
↓ ← ↓ ← A => Special Forward Punch
→ ↓ → ↓ A => Special Uppercut
